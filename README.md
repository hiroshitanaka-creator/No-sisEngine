# NoēsisEngine

> *"They were not disagreeing.
> They were speaking different languages."*

意見には経がある。
その経から、それぞれの歳差運動が生まれる。

軸が違うから、言葉がすれ違う。
対立に見えるものの正体は、
認識の軸の差異だ。

NoēsisEngineは、
表層の言葉（doxa）を哲学的に分解し、
各人の軸を精緻に取り出す。

演繹的な帰着として、
なぜその判断に至ったかが見える。
否決された意見の中に眠る価値が、浮かび上がる。

合意を強制しない。
正解を裁かない。
**軸を明らかにする。それだけだ。**

*Noēsis——臆見を超えた、純粋知性による本質の把握。*
